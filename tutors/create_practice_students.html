Create a Practice Set

Welcome to the Practice Set creation page! Here, you can customize a set of problems or tasks for your students to work on. To get started, please select the student you want to create a practice set for. Once you choose a student, you'll be able to further tailor the problem set based on subject, difficulty, or specific topics the student needs to practice.

Typical steps to create a practice set:
Select a Student
Choose subject(s) and topics
Add or create custom questions
Review and publish the practice set

Click the button below to pick the student for whom you'd like to create this practice set.
